## 経営ジャンルでバズる投稿の法則

```
1. 数字を使う：「月8桁」「2年で」「7店舗」
2. 逆張りタイトル：「やらなかったこと」「やめてよかった」
3. 断言型：「〜すべき理由3つ」「〜は間違い」
4. 経験談ベース：リアルな失敗・成功ストーリー
5. 短文×余白：1文を短く、改行多め
```

## 自社への応用ポイント

- **発信軸**：「SNS運用代行×美容経営」という専門ポジション
