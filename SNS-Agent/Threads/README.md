# Threads AIワークスペース

Maison HALUおよび今後のSNS運用代行クライアントのThreads運用を効率化するワークスペースです。

---

## フォルダ構成

```
Threads/
├── README.md               このファイル
├── competitors/            競合・参考アカウント分析
│   └── 競合分析テンプレート.md
├── posts/                  投稿管理
│   └── 投稿作成テンプレート.md
├── analysis/               データ分析・レポート
│   └── 週次分析テンプレート.md
└── prompts/                AI投稿生成プロンプト集
    └── 投稿生成プロンプト.md
```

---

## テンプレートの使い方

### 競合分析
1. `competitors/競合分析テンプレート.md` をコピー
2. ファイル名を `競合アカウント名_YYYY-MM.md` に変更
3. 調査内容を記入

### 投稿作成
1. `posts/投稿作成テンプレート.md` をコピー
2. ファイル名を `YYYY-MM-DD_投稿テーマ.md` に変更
3. 投稿文を記入し、投稿後に結果を記録

### 週次分析
1. `analysis/週次分析テンプレート.md` をコピー
2. ファイル名を `週次レポート_YYYY-MM-DD.md` に変更
3. 毎週月曜日に前週のデータを記入

### AI投稿生成
1. `prompts/投稿生成プロンプト.md` から目的に合うプロンプトを選ぶ
2. Claude Codeに貼り付けて必要事項を埋める
3. 生成された投稿を `posts/` に保存

---

## 運用フロー

```
月曜：週次分析レポート作成（analysis/）
火〜木：投稿文作成（prompts/ → posts/）
金：来週分の投稿を予約・準備
随時：競合チェック（competitors/）
```

---

## クライアント管理

複数クライアントを管理する場合は `Clients/` フォルダ配下に
クライアント名のフォルダを作成し、同じ構成を複製してください。

```
Clients/
└── MaisonHALU/
    ├── competitors/
    ├── posts/
    ├── analysis/
    └── prompts/
```
