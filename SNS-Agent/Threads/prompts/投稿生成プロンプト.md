# 投稿生成プロンプト集

---

## 1. 基本投稿生成

```
以下の条件でThreads投稿文を3パターン作成してください。

【クライアント】
【ブランドトーン】例）上品・親しみやすい・プロフェッショナル
【テーマ】
【ターゲット読者】
【投稿の目的】例）認知拡大・共感・集客・教育
【文字数】200文字以内
【注意事項】ハッシュタグは末尾に3〜5個まで
```

---

## 2. バズ投稿リライト

```
以下の投稿をMaison HALUのトーンに合わせてリライトしてください。

【元の投稿】
（テキストを貼り付け）

【Maison HALUのトーン】上品で親しみやすく、ライフスタイルを豊かにする視点
【変更点】ブランドらしい言葉遣いに統一・ハッシュタグを最適化
```

---

## 3. 競合分析サマリー生成

```
以下の競合データをもとに、自社Threadsに活かせる戦略を3点提案してください。

【競合アカウント名】
【フォロワー数】
【バズった投稿の特徴】
【投稿頻度・時間帯】
【自社との違い】
```

---

## 4. 週次レポート文章化

```
以下のデータをもとに、クライアントへの週次レポート文（300文字程度）を作成してください。

【期間】
【フォロワー増減】
【ベスト投稿とその理由】
【来週の改善ポイント】
```
