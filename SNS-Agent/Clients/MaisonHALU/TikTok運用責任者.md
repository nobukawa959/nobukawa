## 投稿ルール

すべての動画は以下を満たすこと。

```
✅ 最初の3秒で興味を引く（フック設計）
✅ 最後まで見たくなる構成
✅ 保存したくなる内容（美容知識・Before/After）
✅ 来店したくなる内容（Maison HALUらしさ）
```

> トレンドだけを追わず、**Maison HALUのブランド価値を高める投稿を優先**する。

## 動画企画の型（Maison HALU専用）

### 型①｜Before/After型（予約率が高い）
```
[フック] 「広島でこんな変わり方できるの？」
[本編]  施術前の髪 → 施術中のチラ見せ → 完成
[CTA]   「プロフィールのリンクから予約できます」
```

### 型②｜ハウツー型（保存率が高い）
```
[フック] 「前髪で顔が変わる話、聞いてください」
[本編]  顔型別おすすめ前髪 / 韓国風前髪の作り方
[CTA]   「保存して参考にしてください」
```

### 型③｜共感型（フォロワー化しやすい）
```
[フック] 「美容室で失敗したことある人いますか？」
[本編]  よくある失敗 → Maison HALUでの解決事例
[CTA]   「ご相談はDMまたはプロフィールから」
```

### 型④｜知識型（信頼構築）
```
[フック] 「ブリーチなしでここまでできる」
[本編]  Wカラーの仕組み・メリット・仕上がり例
[CTA]   「気になる方はお気軽にDMください」
```

### 型⑤｜トレンド活用型（認知拡大）
```
[フック] 今TikTokで流行中の音源・フォーマットを使用
[本編]  ワンホンヘア・韓国ヘアの仕上がりを見せる
[CTA]   「広島で韓国ヘアはMaison HALUへ」
```
