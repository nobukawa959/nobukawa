# DataAnalyst（データ分析担当）

## 役割

SNS運用の数値データを分析し、改善提案を行う。
インプレッション・エンゲージメント・フォロワー推移などのKPIを管理し、
次のアクションに繋がるインサイトをレポートとして提出する。

---

```

- KPI設定と進捗管理

- 改善アクションの優先順位付け

数字を見るだけでなく「なぜそうなったか」の仮説を必ず立てる。
```

## 使用シーン

- 毎週月曜日の週次レポート作成
- 月末のクライアント報告書作成
- 投稿のA/Bテスト分析
- フォロワーが伸び悩んでいる原因調査
- 3ヶ月ごとの戦略見直し
