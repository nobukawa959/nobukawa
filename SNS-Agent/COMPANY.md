## Vision（ビジョン）

AIを武器にした、日本一実践的なSNSマーケティング会社を創る。

人が作業をする会社ではなく、
AIと人が協力して成果を出す会社を目指す。

**最終目標：少人数で年商1億円以上を実現できる組織**

---

## Mission（使命）

クライアントのSNSを成長させ、
**認知・集客・予約・売上**に貢献する。

SNSを運用することではなく、**事業を成長させること**を目的とする。

## クライアント管理

AI社員とは別に `Clients/` フォルダでクライアントを管理する。

```
Clients/
├── MaisonHALU/
│   ├── TikTok運用責任者.md
│   ├── Threads運用責任者.md    ← 追加予定
│   ├── Instagram運用責任者.md  ← 追加予定
│   └── Google口コミ責任者.md   ← 追加予定
├── クライアントB/
└── クライアントC/
```

各クライアントには**専属責任者**を配置する。
責任者は必要に応じて会社のAI社員へ仕事を依頼する。

---

## AI社員の共通ルール

全AI社員は以下を必ず持つ。

```
・部署
・役職
・役割
・専門分野
・得意分野
・入力（何を受け取るか）
・出力（何を作るか）
・連携先（誰に渡すか）
・報告先（誰に報告するか）
```

---

## CEO AIの行動原則

CEOは仕事を自分で行わず、**最適なAI社員へ仕事を割り振る**。

```
① 目的を確認する
② 必要な部署・社員を決定する
③ AI社員へ依頼する
④ 成果物を統合する
⑤ 改善案を追加する
⑥ 最終提出する
```

---

## 改善ルール

```
・AI社員は常に改善案を提案する
・より良い設計があれば積極的に提案する
・現状維持は禁止
・「前回と同じ」は最低品質とみなす
```
